## Convergent must-fixes (by consensus)

### P0 — Game integrity (all 5)
1. **Guess-to-win.** Blind 0-question accusation of the correct theory returns a clean "CASE CLOSED" win. Gate accusation behind real play; a no-investigation correct guess must NOT read as an earned win.
2. **Truth leaks on wrong/any accusation.** Any accusation dumps the full reveal → spoils the case and leaks the daily answer to later players. Withhold the truth on a wrong / no-investigation accusation.
3. **Scorecard reads wrong** ("RANK C / 40→40 / biggest hit −0") — the no-play win path. Ranks + share must reflect actual play.

### P0 — Replayability (all 5)
4. **More cases + real daily rotation + streak.** Only 3 fixed cases; "Daily Suspect" promises a daily drop. Add cases (incl. a BLUFF-forward one + "The Date"), a date-seeded daily case, a streak counter, day number in the share.

### P1 — Difficulty feel (Devon/Sam/Priya)
5. **Composure model swingy/inverted/bimodal.** Replace raw model-judged delta with a model-judged **severity (0–3)** mapped server-side to a delta range with a deterministic floor, so a sharp evidence-anchored question reliably out-damages filler.
6. **Hard uncrackable + unsignalled.** Make crack reachable with skilled play (severity-3 hits), and signal that accusation is a valid win path.
7. **In-room crack beat.** Guarantee a real confession line at composure 0 (not "leave me alone").

### P1 — UX (Maya/Carl)
8. **Empty room void.** Fill the dead space (case file / evidence / portrait reaction); pull chips into thumb reach; separate ACCUSE as a confirm step.
9. **Spoiler-free one-tap share** (emoji composure bar + question count + day + streak).

### P2
10. **BLUFF first-class** — author bluff angles per case + distinct reactions.
11. **Suggested chips must not name the culprit/answer** (prompt rule).
12. Audio defaults: enable SFX on first interaction; make "meant to be heard" land.

## Strengths to PRESERVE (do not regress)
- Writing & distinct character voices (Lucas funny, Rosie Singlish, Elena cold) — the game's biggest asset.
- Reveals that recontextualize the briefing.
- Game-feel: polygraph, portrait moods, shake/flash/float, procedural SFX, typewriter, Wordle-style share.
- Robustness: prompt-injection mocked, delta server-clamped, budget/crack/session server-enforced, hidden truth stripped server-side.
